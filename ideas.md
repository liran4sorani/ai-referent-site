## הכיוון הנבחר: "אמון מוסדי" — Trust Architecture

### Design Movement
**Corporate Fintech Modernism** — ממשק שמשדר אמינות מוסדית עם נגיעות של חדשנות טכנולוגית. בהשראת אתרי FinTech מובילים כמו Stripe, Brex ו-Plaid — אך עם זהות ישראלית ברורה.

### Core Principles
1. **אמינות מעל הכל** — כל אלמנט עיצובי מחזק את תחושת הביטחון והמקצועיות
2. **בהירות מידע** — מידע מורכב מוצג בצורה פשוטה וברורה
3. **RTL מושלם** — עיצוב שנולד עברי, לא מתורגם
4. **עומק ומרקם** — שכבות עדינות של shadow, gradient ו-blur יוצרות תחושת עומק

### Color Philosophy
- **Navy Primary** `#1a2b5e` — אמינות, עומק, מקצועיות פיננסית
- **Gold Accent** `#f5a623` — ערך, פרמיום, הדגשה
- **Off-White** `#f8f9fc` — ניקיון, מרחב נשימה
- **Slate Gray** `#64748b` — טקסט משני, עדינות

### Layout Paradigm
אסימטרי ומבוסס-עמודות: Hero section עם תמונה בצד ימין וטקסט בצד שמאל (RTL). Cards בגריד 3 עמודות. Section dividers גלים עדינים.

### Signature Elements
1. **Shield + Chat bubble** — הלוגו כמוטיב חוזר בכל הסקשנים
2. **Gradient cards** עם border זהב עדין ו-shadow עמוק
3. **Hebrew chat bubbles** — הדגמה ויזואלית של ממשק ה-AI

### Interaction Philosophy
Smooth scroll, fade-in on scroll, hover states עם scale עדין. כפתורים עם ripple effect. כל אינטראקציה מרגישה מדויקת ומכוונת.

### Animation
- Fade + slide-up (60ms stagger) לכרטיסי features
- Hero text entrance: 400ms ease-out
- Scroll-triggered counters לסטטיסטיקות
- Nav: backdrop-blur on scroll

### Typography System
- **Headlines**: `Heebo` — עברי מודרני, bold (700/800)
- **Body**: `Assistant` — קריא, נעים, professional (400/500)
- **Accent**: `Heebo` 300 — כותרות משנה עדינות

### Brand Essence
**AI Referent** — הרפרנט הוירטואלי שכל סוכן ביטוח צריך. לסוכנים שרוצים תשובות מהירות, מדויקות ומבוססות.
- **Adjectives**: אמין, חכם, מהיר

### Brand Voice
- Headlines: ישיר, מקצועי, בטוח בעצמו
- Example 1: "הרפרנט שלך זמין 24/7 — בלי להמתין, בלי לחפש"
- Example 2: "תשובות מבוססות פוליסה. לא ניחושים."

### Signature Brand Color
**Navy** `#1a2b5e` — הצבע שמזוהה עם AI Referent
